《祁阳一中：2032》
================================================
【简介】
本软件以祁阳一中一百多年的校史为背景，虚构了2032年120周年校庆的筹备工作，
旨在引领用户了解校史，从而达到教育目的。

【操作指南】
Windows：
 推进对话：鼠标左键 / 空格键 / Enter 键
-菜单：按 Esc 或鼠标右键打开

Android：
点击屏幕操作

【配置要求】
Windows 平台：
最低配置
- 操作系统：Windows 10（64位）
- 处理器：双核 CPU @ 1.5 GHz 或更高
- 内存：2 GB RAM
- 显卡：支持 OpenGL 2.0 或 DirectX 9.0c
- 存储空间：600 MB 可用空间

推荐配置
- 操作系统：Windows 10/11（64位）
- 处理器：Intel Core i3 / AMD 同级别或更高
- 内存：4 GB RAM 或更高
- 显卡：支持 OpenGL 3.0+ 的集成或独立显卡
- 存储空间：1 GB 可用空间

Android 平台：
最低配置
- 操作系统：Android 7.0+（API 21）
- 处理器：四核 1.2 GHz 或更高
- 内存：2 GB RAM
- 显卡：支持 OpenGL ES 2.0
- 存储空间：600 MB 可用空间
- 屏幕分辨率：1280x720 或更低

推荐配置
- 操作系统：Android 10+（API 29+）
- 处理器：八核 2.0 GHz 或更高
- 内存：3 GB RAM 或更高
- 显卡：支持 OpenGL ES 3.0+
- 存储空间：1 GB 可用空间
- 屏幕分辨率：1920x1080 或自适应
